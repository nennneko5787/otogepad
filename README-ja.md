# OtogePad

[English](README.md) | 日本語  

音ゲーのコントローラーをあなたのiPadで！

## ゲーム
### 対応済み
- IIDX

### 計画中
- ノスタルジア

### 実装予定なし
- CHUNITHM
- SDVX

## 動かし方
### 必要なもの
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

### 実行
```
git clone https://github.com/nennneko5787/otogepad.git
cd otogepad
uv sync
uv run python main.py
```
